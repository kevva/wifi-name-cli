# wifi-name-cli [![Build Status](https://travis-ci.org/kevva/wifi-name-cli.svg?branch=master)](https://travis-ci.org/kevva/wifi-name-cli)

> Get current wifi name


## Install

```
$ npm install --global wifi-name-cli
```


## Usage

```
$ wifi-name --help

  Usage
    $ wifi-name
    wu-tang lan
```
